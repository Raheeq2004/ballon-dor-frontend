# Ballon d'Or Voting System Frontend

## Project Overview

This project is the frontend side of the Ballon d'Or Voting System developed for the Special Topics in Computer Science course.

The application allows users to:

- Register and login
- Browse Ballon d'Or categories
- View nominees
- View nominee profiles
- Vote for nominees
- View live voting results
- Access admin dashboard features

The project is developed using React.js and Vite with responsive modern UI design.

---

# Technologies Used

- React.js
- Vite
- React Router DOM
- Bootstrap
- CSS3
- JavaScript (ES6)
